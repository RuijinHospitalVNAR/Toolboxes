VLPIM Web Services

VLPIM: A Comprehensive Tool for Immunogenicity Modulation of Virus-like Particles

An integrated computational workflow for modulating protein immunogenicity through epitope identification, MHC-II binding evaluation, and structural superposition.

Features:
  • Epitope identification using NetMHCIIpan
  • Immunogenicity analysis with Sim scores
  • Structural superposition with RMSD and TM-align
  • Interactive web interface

Circular Contact Map

Nanobody–antigen interface analysis

Generate circular ribbon contact maps from mmCIF or edges.csv to visualize residue–residue contacts.

Features:
  • Contact extraction from mmCIF (two chains)
  • Interactive chord diagram (HTML) or PNG export
  • Simple CSV input: source,target[,distance]
